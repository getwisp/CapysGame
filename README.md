## ğŸ¯ Quick Start Guide

1. **Gather Resources**: Click the "Gather Grass" and "Gather Reeds" buttons
2. **Assign Jobs**: Go to the Capybaras tab and assign workers to Grazer and Gatherer jobs
3. **Build Storage**: Build Grass Piles and Reed Bundles for more storage
4. **Expand Housing**: Build Reed Shelters to increase population capacity
5. **Discover Hot Springs**: Play long enough and you'll discover a natural hot spring!
6. **Keep Warm**: Monitor your warmth level - cold capybaras are sad capybaras
7. **Research**: Once you build a Council Circle, unlock the tech tree
8. **Migrate**: When you've built a Great Hall, you can perform The Great Migration for permanent bonuses
